## Features

### Game Upload

The user can upload a game by copy-pasting the game log into a textarea and submitting it. After a successful upload, the user is redirected to the game overview page.

### Game Overview Page

After uploading, the user is shown a game overview page that displays:
- The names of both players
- Which player won the game

## How the user interacts with the website

1. The user plays a game of Star Realms on the Star Realms app (available on mobile or desktop)
2. When the game ends, before returning to the main menu, the user copies the game log to their clipboard (using the in-app feature)
3. The user navigates to the website and pastes the log into the upload textarea
4. After submitting, the user is shown the game overview page with the results
